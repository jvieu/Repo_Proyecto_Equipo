# Base de datos: Viralización en TikTok y charts musicales (2024)

## Autor y publicación

Los datos a utilizar en este proyecto provienen de:

- Spotify Charts (https://charts.spotify.com/)
- Playlists virales de Spotify (Viral 50)
- Recopilaciones de tendencias en TikTok a partir de medios digitales

## Contenido

La base de datos contiene información sobre canciones presentes en charts globales durante 2024, incluyendo:

- Nombre de la canción y artista
- Posición en el ranking
- Número de reproducciones
- Identificación de viralización en TikTok
- Fecha aproximada de viralización
- Variables derivadas como tiempo de entrada al chart, posición máxima y permanencia

Los datos se presentan en formato CSV y corresponden a registros diarios de rankings musicales.

## Pertinencia

Esta base de datos permite analizar la relación entre la viralización en TikTok y el rendimiento de canciones en charts globales, facilitando la identificación de patrones en el comportamiento de canciones virales.

Es fundamental para contrastar la hipótesis del proyecto y construir visualizaciones comparativas.

## Metodología

Los datos de charts serán obtenidos mediante descarga directa desde Spotify Charts.

La identificación de canciones virales se realizará a través de:

- revisión de playlists virales
- análisis de tendencias documentadas en medios
- recopilación manual de información

Posteriormente, se integrarán ambas fuentes en una base de datos unificada, generando variables derivadas que permitan el análisis comparativo.
